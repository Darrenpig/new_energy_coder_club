# Gitee 注册流程和申请加入

![](output_wiki%5Cstatic/Qx4ybFvJwobL6rxTYp8c4obGn7b.png)

1.Gitee 搜索，并点击 [https://gitee.com/](https://gitee.com/)官网

2.注册，并绑定微信

![](output_wiki%5Cstatic/NfCubzWOgoBR52xGjoXcFLo4n0f.png)

3.Star 仓库 [https://gitee.com/darrenpig/new_energy_coder_club](https://gitee.com/darrenpig/new_energy_coder_club)

![](output_wiki%5Cstatic/DIKYbU7KZoSsFDxCxRscBy5fn2c.png)

[https://gitee.com/login?redirect_to_url=/darrenpig/new_energy_coder_club?apply=1](https://gitee.com/login?redirect_to_url=/darrenpig/new_energy_coder_club?apply=1)

然后加下 maintainer,让他拉你进大群（开发交流群）

![](output_wiki%5Cstatic/WUxsbNz4joauVfx2qi2cPZCBnDA.png)

4.Fork 我们的项目

![](output_wiki%5Cstatic/J296bEc7boE6fLxblx0ct8y6njc.png)

然后就可以编辑啦

5.当然可以提交 Pr，上传自己感兴趣的项目，交给大家一起维护与交流进度
