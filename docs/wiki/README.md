## 🏆 NEC 团队简介

**NEC 开源仓库团队**（New Energy × Robotics）是一支依托于全国大学生机器人大赛 ROBOCON 建立的大学生创新实践团队，成立于 2022 年。

### 核心特色
- 🔬 **双驱动模式**: 竞赛驱动 + 项目驱动
- 🤖 **核心技术**: 机器人、嵌入式、AI 视觉
- 📖 **开源协作**: Gitee + GitHub 双平台同步
- 🎯 **2026 目标**: 冲击 ROBOCON 全国 16 强，争取国际赛资格

### 团队架构
- 🏆 竞赛项目: ROBOCON、AIC 工业视觉等
- 🔬 科研项目: 人形机器人、星闪技术等
- 👥 运营团队: 项目管理、开源运营、财务合规
