# NEC Quickstart Demo（无硬件最小可运行）

> 目的：让每个新人 5-10 分钟内跑起来一个“能看到结果”的 Demo。

## 运行

在仓库根目录执行：

```bash
python docs/nec_quickstart/hello_nec.py --seconds 20
```

输出：

* 控制台：运行摘要 + ASCII 轨迹图
* 文件：`docs/nec_quickstart/out/run_log.csv`

## 常见问题

* macOS 若 `python` 不存在，用 `python3`
* Windows 注意：Python 安装时勾选 “Add to PATH”
