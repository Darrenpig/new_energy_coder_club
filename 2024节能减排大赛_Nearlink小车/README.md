#### 项目介绍
- **标题**：源头治理-智慧化工
- **简短描述**：低成本、节能的工厂巡检方案
- **长描述及官网地址**：全国大学生节能减排竞赛（能动类）官网

#### 建仓目标
- **主要方向**：化工设计（节能减排绿色工厂设计）
- **特色**：加入四轮工厂巡检小车（光伏、节能运维向）
