## 一、硬件架构
### 1. 核心控制器
- **MEGA 2560** (主控单元)  
- **ESP32-S3** ×3（底盘运动控制）  
- **STM32** ×2（ROS扩展板，12V驱动）

- **传感器阵列**  
  ▸ 24路舵机驱动板 ×2（PS2协议）
