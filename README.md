## About The Project

本分支为工业设计小组量身定制，旨在提供更符合设计师逻辑的项目结构和工作流程。在这里，你可以找到与工业设计相关的资源、工具和项目文档。

### Built With

* [Figma](https://www.figma.com) - UI设计工具
* [Sketch](https://www.sketch.com) - UI设计工具
* [Adobe XD](https://www.adobe.com/products/xd.html) - UI设计工具
* [Blender](https://www.blender.org) - 3D建模工具
* [AutoCAD](https://www.autodesk.com/products/autocad/overview) - CAD设计工具

## Getting Started

### Prerequisites

* 安装设计工具
  ```sh
  npm install figma sketch adobe-xd blender autocad
  ```

### Installation

1. 克隆仓库
   ```sh
   git clone https://github.com/your_username/industrial_design_branch.git
   ```
2. 安装依赖
   ```sh
   npm install
   ```
