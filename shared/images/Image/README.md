## 📖 使用指南

### 🔗 引用方式
在README或其他文档中引用图片时，请使用相对路径：
```markdown
![Logo](./shared/images/Image/Logo.png)
![Roadmap](./shared/images/Image/Roadmap.png)
```

### 📏 图片规格建议
- **Logo**: 建议使用SVG格式，支持缩放
- **展示图片**: 推荐1920x1080或16:9比例
- **技术图表**: 确保文字清晰可读
- **团队照片**: 高清原图，便于后续处理

### 🎨 设计规范
- 保持品牌色彩一致性
- 使用统一的字体风格
- 图片命名采用中文，简洁明了
- 重要图片提供多种格式（PNG、SVG、JPG）
