# 机构 SIG (Mechanical & Control SIG)

## 一、简介
机构 SIG 是 NEC 社区的核心硬件研发力量，涵盖 **机械设计**、**电子控制** 与 **场务保障** 三大职能。
我们致力于打破传统的“机电分家”模式，推动 **机电深度融合**，共同承担机器人全生命周期的研发与交付。

## 三、核心文化
> **"不需要有明确的责任边界，做一种文化，做成一个高凝聚力的组织。"**

- **全栈视野**：鼓励机械成员了解电控逻辑，电控成员熟悉机械结构。
- **共同交付**：所有成员对机器人的最终表现负责，而非仅对自己负责的模块负责。
- **敏捷迭代**：快速验证，快速失败，快速改进。

## 四、协作机制
- **统一规划**：采用 `ROB_CTRL` 统一文件结构，实现机电文件同源管理。
- **每日站会**：每日同步进度，快速解决接口冲突与技术堵点。
- **文档驱动**：所有设计决策、接口定义与变更必须有文档记录（README/Issue）。

### 一、 核心得分动作拆解与机构职责

根据比赛流程，得分动作主要分布在三个区域：武馆（一区）、梅林（二区）和对抗区（三区）。

#### 1. 兵器组装动作（武馆区）
*   **动作描述**：R1取长杆，R2取端头，双方协作将端头装在长杆上形成“兵器”。
*   **涉及机构**：
    *   **机构1组（气动夹爪）**：负责R1对长杆的抓取以及R2对端头的精准拾取。
    *   **机构2/9组（关节电机）**：提供机械臂的自由度，完成长杆与端头的对准与插接动作。
    *   **机构4组（直线导轨）**：辅助机械臂在特定行程内平稳移动，确保对接精度。

#### 2. 武术秘籍（KFS）收集动作（梅林区）
*   **动作描述**：R1在通道收集R1 KFS；R2在树林方块上识别并收集R2 KFS。
*   **涉及机构**：
    *   **机构1组（气动夹爪）**：执行抓取秘籍纸箱（350mm立方体）的动作。
    *   **机构3/5组（底盘机构）**：R1通过通道，R2跨越或避开高低不同的梅林方块。由于梅林方块高低不平，底盘需要具备一定的越障或精准定位能力。
    *   **机构7组（电机与布线控制）**：在复杂的梅林环境中，确保传感器（识别秘籍图案）与驱动电机的协同。

#### 3. 九宫格投放与对抗动作（对抗区）
*   **动作描述**：将KFS投放到九宫格的不同层级，或使用兵器移除对方的KFS。
*   **涉及机构**：
    *   **机构4组（直线导轨）**：九宫格高度达2020mm，直线导轨是实现KFS垂直升降至中层（R2负责）或顶层（R1举起R2后负责）的关键。
    *   **机构2/9组（关节电机）**：执行“举起对方”的重负载动作，以及使用兵器横向拨除对方秘籍的动作。
    *   **机构8组（车架）**：作为承重主体，必须支撑R1举起R2时的重心偏移和结构强度。

### 二、 机构分组任务细化表

各组针对得分任务的开发重点如下：

| 机构分组 | 核心任务 / 模块 | 对应得分动作与规则要求 |
| :--- | :--- | :--- |
| **机构1组** | 气动夹爪 | **抓取多元物料**：需兼容直径约20mm的长杆、不规则形状的端头（拳、掌、矛尖） |
| **机构2/9组** | 关节电机 (DM/AK系列) | **高动态协作**：完成R1/R2对接动作；在对抗区提供举起R2或挥动兵器所需的扭矩。 |
| **机构3/5组** | 舵轮 / 全向轮底盘 | **全向移动与越障**：R1需在狭窄通道快速穿梭，R2需在梅林方块间自主规划路径并移动。 |
| **机构4组** | 直线导轨 | **垂直高度覆盖**：负责将秘籍提升至九宫格中层（约1000mm+）和顶层（约1500mm+）及350mm的秘籍纸箱。 |
| **机构6组** | 3508+N630驱动方案 | **底盘/大功率驱动**：为底盘移动或重型提升机构提供动力保障。 |
| **机构7组** | 电机控制+布线 | **系统稳定性**：处理R2自主识别秘籍（甲骨文/小篆）后的执行机构反馈。 |
| **机构8组** | 车架+方管 | **结构刚度**：确保R1在举起R2（两车总重限50kg）时，车体不翻覆且机构不形变。 |
| **机构10组** | 宇树8010+四足结构 | 轮足跳跃：进行相关轮足机构设计 |

### 三、 关键动作的技术挑战提示
*   **协作精度**：组装兵器时，规则严禁R1与R2发生直接肢体接触，所有动力必须通过长杆和端头传递，这对**机构2/4/9组**的控制精度要求极高。
*   **负载能力**：若要争夺“武术大师”称号，R1必须能举起R2投放顶层KFS，这要求**机构8组**的车架和**机构2/9组**的升降关节具备极强的瞬时负载能力。
*   **防守机制**：规则允许使用本队部件或KFS挡住格子以防止对方移除，**机构1组和2组**可开发特定的“防守姿态”机构。

*注：以上分析完全基于提供的ROBOCON 2026 赛事规则及图册资料。*
