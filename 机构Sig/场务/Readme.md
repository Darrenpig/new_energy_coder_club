# 场务组

## 一、背景与定位
- 面向 CURC-ROBOCON 赛季的训练与比赛现场保障，负责场地搭建、设备上电与联调、值班与记录、安全与应急、物资与档案管理。
- 与机械、电控紧密协作，保障“能正常训练/能按时比赛”的最低可用现场条件（Minimum Viable Field）。

## 二、工作范围模块
- 场地搭建与维护：尺寸校核、标线粘贴、障碍/装置安装、定期巡检与修复
- 通电与网络：供电拓扑、UPS/防护、交换机与内网规划、SSID/有线编号
- 设备联调支持：相机/灯光/网关/打标/地标、与电控/视觉联动测试
- 值班与现场秩序：进场/离场登记、设备借还、训练排班与冲突协调
- 安全与合规：动火/高压/旋转件管控、PPE 配备、事故上报与复盘
- 物资与档案：物资台账、耗材补给、场地变更记录、版本标识

## 三、核心职责
- 保障训练/比赛日场地可用性与安全合规
- 提供入场即用的基础设施（电源、网络、标线/地标）
- 形成标准化 SOP 与可追溯记录，支撑多人协作与交接

## 四、现场 SOP
- 训练日开场
  - 到场签到 → 设备巡检 → 通电与网络检查 → 标线/装置完整性确认 → 开放训练
- 训练日收场
  - 关机断电 → 整理回位 → 借还核对 → 异常登记与拍照 → 提交日报
- 比赛日前一天（-1D）
  - 全量校核尺寸/装置 → 备份物资 → 设备健康检查 → 网络与供电压测 → 发布赛日安排
- 比赛日
  - 场地快检 → 时段值班 → 异常快速处置与上报 → 赛后复盘与影像归档
- 突发事件
  - 人身安全优先 → 立即断电/隔离 → 通知负责人 → 依据预案处置与记录

## 五、物资与档案
- 物资台账：[416各位置明细 (飞书 Base)](https://scn0bdoc8zxg.feishu.cn/base/E5pEbt6fJalkUvstfmJc23kongd?from=from_copylink) - 包含名称/数量/状态/位置/责任人/采购链接
- 常备清单：胶带/标线/扎带/工具箱/测量尺/标签纸/PPE/备用电源/交换机/网线
- 档案记录：场地尺寸表、装置安装图、变更记录、巡检与维修日志、影像索引
- 命名规范：`YYMMDD_field_check_<area>.md`、`YYMMDD_issue_<topic>.md`

## 六、人员与培训
- 角色分工：场务负责人、值班负责人、物资管理员、网络/电源维护
- 排班机制：周排班 + 比赛日专项安排，冲突提前 24h 协调
- 培训内容：安全规范、SOP 操作、测量与标线方法、供电与网络基础

## 七、安全与合规
- 高压/旋转件/动火作业须审批与监护，佩戴 PPE
- 供电规范：分路标识、过载保护、绝缘与防水，UPS 供关键设备
- 网络规范：VLAN/口标识/线缆标签，禁止私接路由与热点污染
- 事故处理：先控风险后上报，形成文字与影像证据，复盘改进项闭环

## 八、沟通与协作
- 与机械/电控/视觉在每日站会同步场地变更与需求
- 需求提交途径：开源仓库 issue 或群内固定表单
- 重要事项以文字记录留痕，避免口头决策失效

## 九、模板与表单
- 场地巡检清单（尺寸/装置/标线/安全项）
- 值班签到与借还表
- 异常/事故记录表
- 赛日安排与人员排班表

## 十、指标与复盘
- 可用性：训练开场延迟 ≤ 10 分钟，赛日零重大事故
- 完整性：关键装置缺失率 ≤ 1%
- 合规性：安全巡检覆盖率 100%
- 复盘：每周总结问题、措施与责任闭环
