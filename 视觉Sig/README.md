#### 视觉建议学习内容：

##### 2. 基础操作：学会用OpenCV（核心视觉库）读取/显示/保存图像、图像的色彩空间转换（如RGB转HSV，HSV更适合颜色阈值分割）、图像裁剪与缩放。
